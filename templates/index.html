The Best Places To Live In The UK



An interactive map of the UK, showing the best schools, transport links, places to live and more. Still a work in progress, so please add suggestions in the comments below. All data is (to the best of my knowledge) sourced publicly, and is available at on my github page along with the source code for this page and the underlying plot.

Things I'd like to add as I go along include: Affordability metrics, more on transport links, more on good schools, anything else that seems like a good idea.
